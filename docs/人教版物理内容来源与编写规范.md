# 人教版物理内容来源与编写规范

更新日期：2026-07-19
适用版本：`v1.2.0-rc.1`

## 1. 内容基线

物理主线依据教育部《义务教育课程方案和课程标准（2022年版）》确定课程范围，依据人民教育出版社新版初中物理教材介绍核对教材结构、主题覆盖和科学方法要求。

- 教育部课程标准：<https://www.moe.gov.cn/srcsite/A26/s8001/202204/t20220420_619921.html>
- 人民教育出版社新版初中物理教材介绍：<https://www.pep.com.cn/xw/zt/hd/12/xjcjs/cz/202409/t20240925_1995627.html>
- 教育部初中物理实验教学基本目录说明：<https://www.moe.gov.cn/jyb_xwfb/moe_2082/2023/2023_zl24/202311/t20231123_1091839.html>

人教版教材介绍明确：初中物理全套共 22 章，覆盖课程标准 5 个一级主题、18 个二级主题和 89 个三级主题。项目据此建立八年级上册 1 至 6 章、八年级下册 7 至 12 章、九年级全一册 13 至 22 章的稳定目录。

教育部实验教学基本目录共列出 68 项初中物理基本实验，并强调科学规范、安全操作、应急预案和教师组织指导。本项目当前收录的 29 个重点实验是知识讲解子集，不宣称替代基本目录全部实验；每个已收录实验都必须保留目的、器材、步骤、现象、结论、误差和安全说明。

## 2. 章节与规模

| 册次 | 章节 | 章节知识点 | 原创示例 | 重点实验 |
|---|---|---:|---:|---:|
| 八年级上册 | 机械运动、声现象、物态变化、光现象、透镜及其应用、质量与密度 | 26 | 78 | 10 |
| 八年级下册 | 力、运动和力、压强、浮力、功和机械能、简单机械 | 20 | 60 | 9 |
| 九年级全一册 | 内能、内能的利用、电流和电路、电压 电阻、欧姆定律、电功率、生活用电、电与磁、信息的传递、能源与可持续发展 | 38 | 114 | 10 |
| 合计 | 22 章 | 84 | 252 | 29 |

每章至少包含 3 个独立知识点、1 个章节方法模板、1 组公式或规律、4 个题干信号。每个知识点固定包含通俗理解、核心知识、公式或规律、易错提醒和 3 道原创示例；涉及实验时增加目的、器材、步骤、现象、结论、误差与安全事项。

## 3. 数据组织

物理保留两条互补入口：

1. 教材主线：学科 -> 册次 -> 章节 -> 知识点 -> 方法模板。
2. 综合复习：学科 -> 6 个综合专题 -> 复习知识点与方法。

教材数据文件：

- `data/physics-curriculum-grade8-upper.js`
- `data/physics-curriculum-grade8-lower.js`
- `data/physics-curriculum-grade9.js`
- `data/physics-curriculum.js`
- `data/physics-curriculum-builder.js`

稳定 ID 不使用数组下标。章节 ID 使用 `phy-chNN-*`，知识点 ID 使用所属章节前缀，示例和模板也具有独立 ID。搜索、收藏、最近浏览均携带 `subjectId: "physics"` 和内容类型。

## 4. 编写规则

- 概念讲解先说明研究对象、条件和物理过程，再给结论。
- 公式同时说明物理量、适用条件、方向和单位，不只展示字母等式。
- 计算示例统一执行“提取对象与条件 -> 调用规律 -> 单位和常识复核”。
- 作图内容明确法线、箭头、力的作用点、电路节点和正负接线柱等规范。
- 实验内容同时记录控制变量、转换思想、操作顺序、误差方向和安全事项。
- 生活用电、加热、压力容器等内容必须保留安全边界，不引导学生进行危险操作。
- 学生电学实验只使用低压直流电源；家庭 220 V 线路、放射源和核反应只作知识说明，不安排接触、拆装或自行实验。
- 实验观察与理想推理必须分开书写，不能把推理结论伪装成直接实验事实。
- 不把近似条件写成无条件结论，例如浮沉、热效率、声速和 `g` 的取值均说明使用场景。

## 5. 原创与版权边界

教材和课程标准只用于核对课程范围、目录层级和能力要求。项目不复制教材正文、原题、解析、实验照片或原书插图。全部讲解、示例题干、答案、解析、步骤和界面图示均为原创整理；示意图只辅助理解，不能代替文字中的方向、符号和单位。

## 6. 知识图解规范

- 84 个教材主线知识点各使用 1 张独立图解，不复用综合专题的通用图片。
- 源图路径统一为 `assets/figures/generated/subjects/physics/knowledge/<knowledgeId>.png`，尺寸固定为 `1280 x 900`。
- 图中文字只承担标题、公式摘要和图中必要标记，完整概念、条件、单位和实验说明仍由页面结构化文字提供。
- 电路图中电流表串联、电压表并联，导线不得穿过电源、仪表或电阻形成短接；电磁图明确磁场、运动或受力方向。
- 光学图明确主光轴、法线、入射点和箭头；力学图明确研究对象、作用点和方向；热学与能源图明确过程或转化链。
- 源图由 `scripts/generate-physics-knowledge-assets.py` 生成，云端图由 `scripts/prepare-remote-assets.js` 压缩为 `960 x 675`，本轮最大 47.1 KB。
- 84 张压缩图均已上传到正式微信云存储，并逐项通过临时地址验证；云图片加载失败时不影响文字知识内容阅读。

## 7. 自动校验

物理专项检查：

```bash
node scripts/check-physics-curriculum.js
node scripts/check-physics-accuracy.js
node scripts/check-unique-figures.js
```

脚本校验 3 册、22 章、84 个知识点、22 个模板、252 道示例、章节顺序、必填字段、引用关系和稳定 ID，并验证牛顿第一定律、托里拆利、阿基米德原理、伏安法、焦耳定律、电磁感应、光纤等关键词能命中对应知识点。图片校验同时检查 84 张物理知识图的路径、PNG 格式、`1280 x 900` 尺寸和文件内容唯一性。

全局检查：

```bash
node scripts/check-subject-content.js
node scripts/check-release-readiness.js
```

正式上传前还需在微信开发者工具和真机上检查三册切换、章节跳转、知识点、方法模板、搜索、收藏、云图片失败降级和长文本显示。
