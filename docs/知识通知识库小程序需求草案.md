# 知识通知识库小程序需求草案

## 1. 项目定位

- 产品名称：知识通
- 目标平台：微信小程序
- 目标用户：初中学生、家长
- 内容方向：初中数学、英语、物理知识库与题型模板
- 当前阶段：先完成数学 MVP，跑通开发、测试、上传、审核流程，再扩展英语和物理

## 2. 核心目标

- 按人教版知识体系组织内容
- 提供基础知识、典型例题、进阶题、压轴题
- 提供常见题型的解题模板与答题步骤
- 在手机端清晰展示公式、几何图形、表格、图片等非纯文本内容
- 页面结构清晰，便于快速查找知识点和题型

## 3. 第一阶段范围：数学

### 3.1 学段与版本

- 覆盖初中数学
- 以“人教版目录体系”为主线组织内容
- 产品表达建议：按人教版知识结构整理，不直接整页复制教材原文或插图

### 3.2 内容层级

- 学科
- 年级
- 册别
- 章节
- 小节
- 知识点
- 题型
- 题目
- 解法模板

### 3.3 数学内容模块

- 基础知识
- 公式定理
- 易错点
- 典型例题
- 分层练习
- 压轴题
- 解题模板
- 错题与收藏

### 3.4 数学重点展示能力

- 数学公式排版
- 几何图形展示
- 坐标图与函数图简单展示
- 步骤化解题过程
- 题干、图形、答案、解析分区展示

## 4. 后续扩展范围

### 4.1 英语

- 单词与短语
- 语法知识点
- 句型模板
- 完形、阅读、写作模板
- 听力或发音资源预留
- 图片、表格、音频等非文本内容展示

### 4.2 物理

- 概念与规律
- 公式与单位
- 实验题
- 作图题
- 计算题模板
- 电路图、受力图等图示展示

## 5. 小程序信息架构建议

- 首页
- 数学首页
- 年级/册别页
- 章节目录页
- 知识点详情页
- 题型模板页
- 题目练习页
- 收藏页
- 错题本页
- 我的页面
- 搜索页

## 6. MVP 必做功能

- 数学学科入口
- 年级和章节导航
- 知识点详情展示
- 公式/图形展示
- 典型题与解析展示
- 题型模板展示
- 收藏
- 最近浏览
- 搜索

## 7. 第二阶段增强功能

- 错题本
- 学习记录
- 自测练习
- 题目难度筛选
- 按题型筛选
- 夜间阅读样式
- 内容后台导入

## 8. 需要尽早确认的关键事项

- 是否允许离线查看部分内容
- 是否需要账号登录与数据同步
- 是否要做游客模式
- 是否需要搜索全站知识点和题型
- 是否需要按“基础/提升/压轴”筛选
- 是否需要家长视角或学习周报
- 是否需要后续接入 AI 问答

## 9. 风险与注意事项

- “人教版全部内容”如果直接复制教材原文、图片、排版，可能涉及版权风险
- 小程序审核不适合出现占位页、空页面、不可点按钮
- 教育类内容的服务类目和资质，需要在正式提交审核前根据当时微信后台要求确认
- 非文本内容要提前设计统一组件，否则后面扩展英语和物理会反复返工

## 10. 当前建议结论

- 先做“数学知识库 MVP”
- 先把内容模型、页面结构、图文公式组件、搜索和收藏跑通
- 内容先按“人教版结构化整理”实现，不直接照搬教材原文
- 等数学版本稳定后，再平移扩展到英语和物理
