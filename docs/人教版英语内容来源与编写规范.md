# 人教版英语内容来源与编写规范

更新日期：2026-07-22

## 1. 内容定位

英语内容以人民教育出版社新版义务教育英语教材的册次、单元标题与顺序为导航基线，以教育部《义务教育英语课程标准（2022 年版）》为知识边界。项目提供原创的词汇用法、语法解释、双语例句、核心表达和易错提醒，不复刻电子课本。

`verified` 只表示该册的单元标题和顺序已核对，不表示项目中每单元 8 个核心词等同于教材逐页词表。这些词条是按单元主题和初中学习难度筛选的知识讲解，并使用稳定 ID 承载搜索、收藏和迁移。

当前可学习范围：

- 七年级上册：3 个预备单元 + 7 个正式单元。
- 七年级下册：8 个正式单元。
- 八年级上册：8 个正式单元。
- 八年级下册：8 个正式单元。
- 九年级上册：8 个正式单元。
- 九年级下册：新版完整目录公开后录入，当前只显示“待发布”，不建立空单元。

合计 42 个可学习单元、336 个逐词讲解、84 个语法点和 604 条原创例句。七年级上册 80 个词和 20 个语法点已完成补深；人教社介绍说明全套新版教材共 6 册、44 个正式学习单元，本项目当前未把尚未可靠核对的九下 5 个正式单元和 2 个选用戏剧写成猜测内容。

## 4. 每单元最低标准

每个可学习单元固定包含：

1. 单元主题和至少 3 条可直接使用的核心表达。
2. 8 个核心词，每个词都有词性、中文义、具体用法、词形变化、至少 2 个搭配、原创英文例句、对应中文翻译和易错提醒。
3. 2 个单元语法点，每个语法点都有概念解释、至少 2 个结构、2 组原创双语例句与说明、至少 2 个常见错误。
4. 1 张独立单元知识图，概括单元主题、核心表达、重点词汇和两类语法结构。

自动检查命令：

```bash
node scripts/check-english-units.js
node scripts/check-subject-content.js
node scripts/check-unique-figures.js
```

专项检查同时验证 42 个单元图片路径与稳定 `unitId` 一致，416 条词汇例句和 188 条语法例句均不重复，英文原句、中文翻译和语法说明字段完整。

七上补深内容另要求每个词具备英美双 IPA、可数性或及物性、两条搭配释义、两组例句和一组辨析；每个语法点具备使用条件、三种结构变式、易混对比、结构图和三组例句。未补深册次继续按基础字段展示，不增加空字段占位。

语言准确性还必须单独运行：

```bash
node scripts/check-english-accuracy.js
node scripts/check-english-depth.js
```

该检查覆盖教材对齐声明、关键不可数名词、不规则复数、情态动词后的动词形式、现在完成时时间信号、条件句范围和本轮手工复核的重点搭配。

## 5. 语言标准与复核规则

- 以教材常用的英式拼写为主，如 `colour`、`favourite`、`practise`；词典普遍接受的美式变体在用法或易错提醒中说明，不把变体当作错误。
- 补深词条同时展示 UK 与 US 的 IPA；仅记录发音事实，不复制第三方词典释义、例句或音频。
- 词性、可数性、不规则词形、介词搭配和动词补语必须能在词条中直接查到，不只给中文词义。
- 例句必须原创、自然、可独立理解，且中文翻译与英文的时态、数量和语气一致。
- 语法点只在本单元已讲解的范围内给核心表达；如果示例引入新结构，必须同步补足结构、用法和易错说明。
- 现在完成时与一般过去时、`used to do` 与 `be/get used to doing`、第一与第二条件句等高频易混点，同时经过脚本规则和人工复核。

语法交叉核对参考：

- Cambridge Dictionary, English Grammar Today, `Used to`：<https://dictionary.cambridge.org/grammar/british-grammar/used-to>
- Cambridge Dictionary, English Grammar Today, `Past simple or present perfect?`：<https://dictionary.cambridge.org/grammar/british-grammar/past-perfect-simple-or-past->
- British Council LearnEnglish, `Conditionals: zero, first and second`：<https://learnenglish.britishcouncil.org/free-resources/grammar/b1-b2/conditionals-zero-first-second>

## 6. 单元图解规范

- 42 个已核对单元各使用 1 张独立图解，路径统一为 `assets/figures/generated/subjects/english/units/<unitId>.png`。
- 源图尺寸固定为 `1280 x 900`，用主题插图、两类语法结构、核心表达和 4 个关键词建立单元知识地图。
- 问候、家庭、校园、天气、旅行、阅读、文化、太空、音乐等主题使用不同图形语义，不以同一底图只替换标题。
- 图片中的英文结构只作快速提示，完整词性、词形、搭配、例句、翻译和易错点仍由页面文字承载。
- 云端图由 `scripts/prepare-remote-assets.js` 压缩为 `960 x 675`，本轮最大 46.2 KB；42 张图片均已上传正式微信云存储并逐项通过临时地址验证。
- 图片加载失败时显示文字提示，不影响核心表达、词汇和语法内容阅读。

## 7. 原创与版权边界

- 允许记录教材册次、单元标题和课程知识范围，用于导航和教学对照。
- 不复制教材课文、听力文本、原题、答案、解析、对话和插图。
- 词汇释义按常见语言知识原创整理；用法说明、例句、翻译和语法解释均重新编写。
- 不使用教材电子页截图作为运行时内容，不把网络教案或题库整段改写后冒充原创。
- 后续新增九下内容时，必须先核对人教社或正式教材目录，再运行专项检查并记录来源更新时间。
