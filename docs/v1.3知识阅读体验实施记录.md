# v1.3 知识阅读体验实施记录

当前版本：`1.3.0-dev.1`

基线：`v1.2.0-rc.1`，Git 标签对应三科内容准确性复核完成后的冻结提交。

## 一、导航结构

- 数学默认路径为“年级 -> 教材章节 -> 知识点”，专题只在“专题索引”中出现。
- 英语默认路径为“册次 -> 单元 -> 单词与语法”，专题只作为跨单元索引。
- 物理默认路径为“册次 -> 章节 -> 知识点与实验”，专题只作为跨章节索引。
- 教材目录和专题索引使用同页分段切换，不再同时纵向堆叠。
- 数学章节、英语单元、物理章节和知识点均提供父级路径与上一篇、下一篇入口。

## 二、知识点阅读页

默认阅读顺序固定为：

1. 标题与摘要。
2. 核心概念。
3. 关键图解。
4. 公式、英语结构或物理实验。
5. 易错辨析。
6. 方法步骤和典型例子。

完整讲解、方法步骤和例题解析默认折叠，用户按需展开。核心要点、公式、英语例句和物理实验支持复制。页面不加入答题、自测、评分和单元任务。

## 三、兼容约束

- 不修改三科既有稳定内容 ID。
- 收藏、最近浏览、旧数学知识点别名和云图片地址继续兼容。
- 专题数据继续保留，搜索和旧链接仍可进入专题页。
- 云图片加载失败时必须保留完整文字内容。

## 四、内容与搜索

- 数学 89 节、英语 42 单元/336 词/84 语法点、物理 22 章/84 知识点均带内容来源、复核状态和复核日期。
- 数学知识页移除通用训练套话，改为具体事实、公式条件、知识关系和易错边界。
- 章节页移除复习清单，所有页面不显示学完自测、测评任务或单元输出任务。
- 搜索结果按实际相关度决定分组顺序，并支持数学、英语、物理及内容类型筛选。
- 英语单词与单元语法结果可直接滚动到对应卡片，全角公式符号会统一后再检索。

## 五、图片与云资源

- 专题页展示专题封面，并在考点和题干信号后展示英语/物理结构示意图。
- 知识点与例题图片扩大显示区域，数学和英语图注直接对应当前内容。
- 运行时共收集 754 张云图片；压缩目录带 `manifest.json`，记录相对云路径、尺寸、字节数和 SHA-256。
- `check-unique-figures.js` 同时检查路径与文件内容重复；`check-remote-assets.js` 检查漏图、尺寸、200KB 上限、路径和哈希。
- 云图片签名改为小程序端接口优先、云函数兜底，签名失败时保留文字且不抛出页面错误。

## 六、本地阅读

- 本地内容结构版本升级为 4，旧收藏、最近浏览和数学笔记继续迁移稳定 ID。
- 首页显示最后一个知识点的“继续阅读”入口。
- 阅读状态保存滚动位置、完整讲解/方法步骤折叠状态和例题展开状态。
- 每个知识点支持一条 800 字以内的本地笔记和最多 5 个自定义标签，“我的”页汇总笔记。
- 不要求登录，不上传阅读位置、笔记或标签。
