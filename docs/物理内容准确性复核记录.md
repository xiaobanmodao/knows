## 2. 逐章结论

| 章 | 公式与规律复核 | 实验与安全复核 | 状态 |
|---|---|---|---|
| 1 机械运动 | 测量记录改为准确数字、估读数字和单位，避免误写成两个数值相加 | 多次测量只用于减小偶然误差，异常数据先判断再处理 | 通过 |
| 2 声现象 | 空气声速限定为 15 ℃ 附近约 340 m/s；响度比较加入距离和环境条件 | 音叉轻敲且远离耳道 | 通过 |
| 3 物态变化 | 晶体熔化加入一定压强、达到熔点和熔化过程；沸腾加入外界气压不变 | 加热实验增加护目镜、器材朝向和防烫要求 | 通过 |
| 4 光现象 | 反射定律完整写明共面、分居和角相等；折射保留法线与垂直入射边界 | 激光不得照射眼睛、皮肤和反光物，光路低于眼睛 | 通过 |
| 5 透镜及其应用 | 补齐 u=2f 和 u=f 两个成像边界 | 禁止用透镜会聚太阳光，优先使用低温发光体 | 通过 |
| 6 质量与密度 | 密度公式的质量、体积对应同一样品，单位换算保持一致 | 天平、量筒和排水法保留防污染、防气泡要求 | 通过 |
| 7 力 | 弹簧规律限定同一弹簧和弹性限度；g 明确近似取值 | 钩码防坠落，测力计不得超量程 | 通过 |
| 8 运动和力 | 牛顿第一定律区分实验观察与无阻力理想推理 | 小车实验控制同高度释放并设置防坠区域 | 通过 |
| 9 压强 | 液体压强限定静止、密度均匀，h 为自由液面到研究点的竖直深度；流速规律限定适用情形 | 压强计先检查气密性，液体及时擦净 | 通过 |
| 10 浮力 | 阿基米德原理明确浸在流体中和排开体积；浮沉判断限定只受重力和浮力 | 溢水杯加满、排尽多余水并排除气泡 | 通过 |
| 11 功和机械能 | W=Fs 限定力与位移同向，P=Fv 限定同向匀速 | 能量实验保留装置固定和运动区域安全 | 通过 |
| 12 简单机械 | 力臂为支点到力作用线的垂直距离；理想滑轮组写明忽略项；效率小于 100% | 钩码下方不得站人，装置须固定牢靠 | 通过 |
| 13 内能 | Q=cmΔt 限定无相变且比热容近似不变 | 水与食用油实验统一质量、输入和散热条件，并增加防飞溅与断电要求 | 通过 |
| 14 内能的利用 | Q=mq 限定完全燃烧；效率分母明确为实际输入能量 | 不安排学生燃烧或热机危险操作 | 通过 |
| 15 电流和电路 | 元电荷写为大小 e，电子电荷量为 -e；电流规律直接给出串并联关系 | 只使用低压直流电源，改接先断电，禁止接入 220 V | 通过 |
| 16 电压 电阻 | 电阻影响因素比较加入温度和控制变量条件 | 电压、电阻实验使用低压直流电源，防止电阻丝过热 | 通过 |
| 17 欧姆定律 | U、I、R 限定同一导体同一状态；实验结论加入温度近似不变 | 闭合前变阻器置最大，改接断电并防短路 | 通过 |
| 18 电功率 | 焦耳定律与纯电阻 Q=W 的边界分开；额定和实际状态分开 | 小灯泡只用低压电源，略高电压时迅速读数 | 通过 |
| 19 生活用电 | 220 V 为我国低压家庭电路额定电压，总电流公式用于负载估算 | 学生不得接触、拆装、测量或检修真实 220 V 线路，故障交由专业人员 | 通过 |
| 20 电与磁 | 磁感线写明闭合方向；安培定则明确使用右手和螺线管电流方向 | 螺线管和电磁铁低压、短时通电，磁体防夹手 | 通过 |
| 21 信息的传递 | 一般关系写为 v=λf，真空中 c=λf；光纤明确全反射 | 不以图片小字替代频率、波长和传播条件 | 通过 |
| 22 能源与可持续发展 | 区分裂变发电与恒星聚变，能源评价保留全生命周期条件 | 学生不接触放射源，不自行开展辐射或核反应实验，只使用模型、资料或模拟 | 通过 |

## 3. 自动护栏

科学性专项检查命令：

    node scripts/check-physics-accuracy.js

该脚本固定检查以下内容：

1. 22 章关键公式和规律的适用条件。
2. 实验观察、控制变量与理想推理的表达边界。
3. 29 个重点实验的目的、器材、步骤、现象、结论、误差和安全字段。
4. 加热、激光、低压电学、家庭 220 V 和核能内容的安全边界。
5. 不严谨旧表述不得重新进入数据。

## 4. 后续维护

- 修改物理公式、规律、实验或安全文案后，必须同时运行结构校验和科学性专项校验。
- 修改公式摘要后需重新生成对应知识图，确认图片中的公式、方向、符号和单位与页面文字一致。
- 新增实验时应先核对教育部基本实验目录，再补全误差来源和安全事项。
- 发布前仍需在微信开发者工具和真机检查长文本、公式换行、图示方向和云图片降级。
