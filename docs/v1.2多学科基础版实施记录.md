# 知识通 v1.2 多学科基础版实施记录

更新日期：2026-07-19
当前版本：`1.2.0-rc.1`
开发分支：`codex/multisubject-v1.2`

## 1. 版本范围

本版把项目从数学知识库扩展为数学、英语、物理三科基础知识库。内容仍以游客模式本地阅读为主，不包含登录、云同步、在线评分、错题本、听力或语音评测。

统一内容层级如下：

- 数学：年级大包 -> 专题/章节 -> 知识点 -> 题型模板。
- 英语：学科大包 -> 教材册次 -> 单元 -> 逐词讲解/单元语法；另保留能力专题。
- 物理：学科大包 -> 教材册次 -> 章节 -> 知识点 -> 方法模板；另保留综合专题入口。
- 通用入口：学科页、专题页、全局搜索、收藏和最近浏览。

## 3. 数据与兼容

通用查询层位于 `utils/subjects.js`，提供：

- `getSubjectHome(subjectId)`
- `getTopicById(subjectId, topicId)`
- `getKnowledgeById(subjectId, knowledgeId)`
- `getTemplateById(subjectId, templateId)`
- `searchAllSubjects(keyword)`
- `getEnglishUnitById(unitId)`
- `getPhysicsChapterById(chapterId)`

英语能力专题与物理综合专题分别位于 `data/english-content.js` 和 `data/physics-content.js`，共享 `data/subject-content-builder.js`。英语教材单元位于 `data/english-units-grade7.js`、`data/english-units-grade8.js`、`data/english-units-grade9.js`，由 `data/english-units.js` 聚合。物理教材章节位于 `data/physics-curriculum-grade8-upper.js`、`data/physics-curriculum-grade8-lower.js`、`data/physics-curriculum-grade9.js`，由 `data/physics-curriculum.js` 聚合。旧数学链接缺少 `subjectId` 时继续按数学解析。

本地内容存储版本为 3。旧收藏和最近浏览会自动补充 `subjectId: "math"`、内容类型和容器信息，并继续解析旧知识点别名。所有内容 ID 在三科范围内全局唯一。

## 4. 页面与交互

- 新增通用学科页 `pages/subject` 和专题页 `pages/topic`。
- 新增英语教材单元页 `pages/english-unit`，展示核心表达、逐词用法、词形、搭配、双语例句、语法结构和易错点。
- 新增物理教材章节页 `pages/physics-chapter`，支持八上、八下、九全切换，展示题干信号、公式与单位、实验数量、章节方法和全部小节。
- 当前版本聚焦知识内容，已删除单元输出任务、学完自测、训练流程、学习目标和通用学习路线等任务型模块。
- 知识点页和模板页支持 `subjectId`，模板也可收藏并写入最近浏览。
- 全局搜索支持全部、数学、英语、物理四种筛选，并按学科和内容类型分组；英语可直接检索单元、单词、搭配和语法。
- 收藏与最近浏览显示学科、类型标识，并按记录类型打开正确页面。
- `content-block` 新增 `example` 与 `experiment` 两种结构。
- “我的”页展示三科实际内容规模，英语显示教材单元、逐词讲解和语法点数量。
