# v1.4 分包底座实施记录

当前版本：`1.4.0-dev.1`

开发分支：`codex/content-packages-v1.4`

冻结基线：提交 `659c532`，标签 `v1.3.0-rc.1`

## 一、普通分包

- 主包保留首页、搜索、收藏、我的、公共组件、存储、云资源解析和 8 个旧路径跳转页。
- `packages/english/` 包含英语目录、单元、专题、知识点、模板及英语数据仓库。
- `packages/physics/` 包含物理目录、章节、专题、知识点、模板及物理数据仓库。
- `packages/math/` 包含数学年级目录、章节、专题、知识点、模板及数学数据仓库。
- 三个分包只依赖主包公共代码，不相互引用；未配置 `preloadRule`。
- `app.js` 启动依赖不再加载任一学科正文，旧数学 ID 迁移改用 89 条轻量别名表。

## 二、统一路由与兼容

- `utils/content-routes.js` 以 `subjectId + type + id + focusId` 生成目标分包路径。
- 首页、搜索、收藏、最近浏览、继续阅读和笔记入口均使用统一路由。
- 原数学、学科、章节、英语单元、物理章节、专题、知识点和模板路径保留为轻量跳转页。
- 旧链接缺少 `subjectId` 时按数学解析；分包导航失败时旧页保留并显示重试入口。
- 本地记录继续保存稳定 ID，不保存页面路径，存储 schema 保持版本 4。

## 三、轻量搜索索引

- `scripts/build-search-index.js` 从三科完整内容构建 `data/search-index.js`。
- 索引共有 833 条：数学 183、英语 492、物理 158。
- 每条只保留稳定引用、标题、短描述、标签和短 token，不包含长例句、完整解析或实验正文。
- 排序保持标题完全匹配、标题前缀、标题包含、别名与关键词命中的顺序。
- 英语单词和语法结果继续使用 `focusId` 直达单元内卡片。

## 四、校验底座

- 分包边界检查确认主包不引用学科正文，分包之间不交叉引用。
- 搜索索引检查确认唯一 key、内容源 SHA-256 和 7 个关键检索词。
- 内容 schema 覆盖 836 个实体、209 个公式块和 35 个实验块。
- v1.3 内容基线包含实体哈希，当前差异报告为新增 0、修改 0、删除 0。
- 发布配置检查已扩展到 15 个分包页面和公共组件。
- 原有数学、英语、物理、资源、准确性、迁移和发布检查继续通过。
