# 数学内容准确性复核记录

更新日期：2026-07-19
适用版本：v1.2.0-rc.1
复核范围：七、八、九年级共 29 个稳定章节容器、89 个教材小节、445 道原创示例、36 条旧链接兼容知识和 36 个以上方法模板。

## 1. 复核依据

- 教育部《义务教育数学课程标准（2022年版）》：<https://www.moe.gov.cn/srcsite/A26/s8001/202204/W020220420582346895190.pdf>
- 教育部《2024年义务教育国家课程教学用书目录》：<https://www.moe.gov.cn/srcsite/A26/s8001/202408/W020250418502592948423.pdf>
- 人民教育出版社新版初中数学教材介绍：<https://www.pep.com.cn/xw/zt/hd/12/xjcjs/cz/202408/t20240826_1994351.html>

现有 29 章是用于导航和兼容的稳定知识容器，不等同于 2024 新教材逐册原始章序。项目内容为原创整理，不复制教材或教辅正文、题目和插图。

## 3. 架构修正

- 新增 data/math-lesson-formulas.js，为 89 个小节逐一配置公式或结论，不再把同一主题公式无条件复用到整章。
- utils/math.js 只按完整小节标题读取公式，因此包装纸盒不会再显示三视图结论，一元二次方程也不会再显示一次方程公式。
- data/math-lesson-facts.js 逐节维护定义、性质和适用条件。
- 36 条 data/math-data.js 兼容知识继续纳入检查；旧收藏和搜索命中这些内容时仍能获得正确公式和示例。
- 数学页面不包含学完自测、单元输出任务或测评任务。

## 4. 自动校验

    node scripts/check-math-content.js
    node scripts/check-math-accuracy.js

check-math-content.js 检查专题、章节、模板、稳定 ID 和占位内容。check-math-accuracy.js 固定检查：

1. 29 章、89 节、445 道示例和 36 条兼容知识的规模。
2. 每节独立事实和公式映射完整，运行时公式与源数据一致。
3. 分式禁值、判别式、等可能概率、函数定义域、位似中心、坡度角和投影条件。
4. 平行线判定/性质、四分位数、纸盒面积、二次函数面积和反比例工程示例的回归结果。
5. 已删除的不严谨旧表述和测评类文案不得重新进入运行内容。

## 5. 后续维护

- 修改定义、公式、示例或模板后，必须运行结构校验与数学准确性校验。
- 修改知识摘要或题干后，重新生成对应知识图和题目图，并核对公式换行、符号与条件。
- 新版教材完整目录更新只调整显示与映射，不直接破坏稳定 ID。
- 新增知识需继续保持原创表达，并记录官方课程边界；非官方题库、博客和旧产品页不能作为唯一依据。
