# 新版教材目录对照与迁移规则

## 一、正式基线

知识通从 v1.1.0 起以以下公开资料作为内容边界：

- 教育部《义务教育数学课程标准（2022年版）》
  https://www.moe.gov.cn/srcsite/A26/s8001/202204/t20220420_619921.html
- 人民教育出版社《人教版义务教育数学（七～九年级）新教材介绍》
  https://www.pep.com.cn/xw/zt/hd/12/xjcjs/cz/202408/t20240826_1994351.html
- 教育部《2024年义务教育国家课程教学用书目录（根据2022年版课程标准修订）》
  https://www.moe.gov.cn/srcsite/A26/s8001/202408/W020250418502592948423.pdf

人教社旧产品页仍保留依据 2011 年版课标编写的目录，不能单独作为新版章节顺序依据。新版教材按起始年级逐步投入使用，因此每一册正式目录公开后单独核对，不根据非官方汇总页一次性改写六册章节编号。

## 三、稳定 ID 与旧数据

- 章节使用语义 ID，例如 `ch01-rational`，显示章号不参与数据关联。
- 生成知识点使用去除章节编号后的标题哈希作为稳定 `lessonId`。
- v1.0.0 的 `chapterId-lesson-N` 作为 `legacyIds` 保留，旧链接继续可访问。
- 小程序首次运行 v1.1.0 时迁移收藏和最近浏览，重复记录按稳定 ID 合并。
- 专题选择保存在 `knows_math_grade`，默认值为 `grade8`。

## 四、内容和版权门禁

- 每个专题至少包含 4 个考点、题干信号、核心知识与独立封面。
- 讲解、例题和方法均重新组织表达，不复制教材或教辅正文。
- 用户提供的练习本照片只用于确认“专题—考点—考法”层级。
- 专题封面使用原创生成图，不使用教材页面、教辅插图、人物肖像、品牌或水印。
- 九年级新版专题在对应正式目录核对完成后进入 v1.2.0，不混入 v1.1.0。
