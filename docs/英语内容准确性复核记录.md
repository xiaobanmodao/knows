## 2. 复核基线

- 教育部《义务教育课程方案和课程标准（2022 年版）》用于界定初中英语学习范围和素养目标。
- 人民教育出版社《人教版义务教育英语（七～九年级）新教材介绍》用于核对 6 册、44 个正式学习单元的总体结构。
- 已可靠获取的正式教材目录用于核对单元标题和顺序；教材课文、对话、习题和插图不进入运行时内容。
- Cambridge English Grammar Today 与 British Council LearnEnglish 用于交叉核对高频易混语法，不替代人教版的单元编排基线。

## 3. 逐册复核结果

### 七年级上册

- 复核了问候、物品归属、`there be`、人称与 `be`、名词所有格、方位介词、时间介词、情态动词 `can`、频度副词、序数词与日期。
- 不可数名词 `information`、`homework` 和不规则复数 `goose → geese`、`child → children` 已设为自动检查基线。
- 将不够自然的 `parent meeting` 修正为 `parent-teacher meeting`。

### 七年级下册

- 复核了特殊疑问句、可数名词复数、祈使句、`must/have to/can`、频率表达、数量表达、现在进行时、天气句型、一般过去时和故事连接词。
- 将不完整搭配 `clever at` 补成 `be clever at doing sth`，并说明介词后接名词或动名词。
- 将容易造成数性混淆的 `do exercise` 改为 `get some exercise`，区分不可数的“锻炼”和可数的 `grammar exercises`。

### 八年级上册

- 复核了一般过去时、复合不定代词、礼貌请求、使役结构、比较级与最高级、定语从句、感叹句、将来表达、宾语从句和第一条件句。
- 将 `relationship with` 补成可直接套用的 `relationship with sb`，将 `organize a room/time` 改为 `organize your room/time`。
- 核对 `species` 的单复数同形、`one of the + 最高级 + 复数名词` 和 `What/How` 感叹句的名词范围。

### 八年级下册

- 复核了现在完成时的持续、经历、进度和影响用法，以及 `for/since`、`already/yet/just`、`used to`、过去进行时、被动期待结构与评价表达。
- 所有现在完成时语法示例均排除 `yesterday`、`last year`、`ago` 等明确结束的过去时间信号。
- 将 `practised an earthquake drill` 改为 `took part in an earthquake drill`，将 `The storm warning arrived` 改为 `The storm warning was issued`。

### 九年级上册

- 复核了现在完成时、`used to / be used to / get used to`、人物定语从句、`It is + 形容词 + for/of sb to do`、`by doing`、动词补语、被动语态、推测情态动词、非限制性定语从句和条件句。
- 将 `orbit around` 收紧为及物动词搭配 `orbit the Earth`，同时保留名词结构 `an orbit around the Earth` 的区分。
- 将旋律“变得安静”改为更自然的 `becomes softer`。
- `More than a Game` 的语法讲解仅覆盖第一、第二条件句，因此将原先未讲解的第三条件句核心表达替换为第二条件句。

## 4. 自动校验

```bash
node scripts/check-english-units.js
node scripts/check-english-accuracy.js
```

`check-english-units.js` 检查数量、字段、ID、图片路径、搜索和例句唯一性。`check-english-accuracy.js` 检查来源声明、词汇事实、已复核搭配、时态信号、情态动词形式、条件句范围和关键例句。

## 5. 边界与后续规则

- 本轮完成的是当前可学 42 单元的知识准确性复核，不宣称 336 个核心词是教材原始词表的逐词复制。
- 项目不复制教材课文、听力材料、对话、习题、答案或插图；例句和讲解继续保持原创。
- 九年级下册必须在官方完整目录可靠核对后再录入，不从题库、博客或旧版目录猜测。
- 后续每新增一个单元，除通过脚本外，至少手工复核词性、数性、词形、搭配、例句、翻译、语法范围和图中英文。
