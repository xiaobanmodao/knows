# 数学目录与模型来源说明

更新日期：2026-07-19
适用版本：v1.2.0-rc.1

## 1. 官方课程边界

当前数学内容以以下官方资料为正式依据：

- 教育部《义务教育数学课程标准（2022年版）》
  https://www.moe.gov.cn/srcsite/A26/s8001/202204/W020220420582346895190.pdf
- 教育部《2024年义务教育国家课程教学用书目录（根据2022年版课程标准修订）》
  https://www.moe.gov.cn/srcsite/A26/s8001/202408/W020250418502592948423.pdf
- 人民教育出版社《人教版义务教育数学（七～九年级）新教材介绍》
  https://www.pep.com.cn/xw/zt/hd/12/xjcjs/cz/202408/t20240826_1994351.html

人教社官方介绍明确说明，新教材对旧“函数”内容进行了重新编排，并在数据分析中增加四分位数、数据分组等内容。项目已在稳定内容容器内补入这些知识，但不把现有 29 个容器宣称为 2024 新教材的逐册原始章号。

## 2. 目录口径

- 当前 29 章是产品中的稳定知识容器，用于保持导航、收藏、搜索和旧链接兼容。
- ch01-rational 等语义 ID 是数据主键，显示章号不参与数据关联。
- 旧版人教社产品页只用于识别历史标题和兼容旧链接，不能单独决定新版教材顺序。
- 新版教材某一册的官方完整目录可靠公开后，再更新显示顺序、章节映射和别名；不根据第三方目录猜测。
- 每个运行时教材小节都有独立稳定 lessonId；旧的 chapterId-lesson-N 继续作为别名解析。

具体目录迁移规则见《新版教材目录对照与迁移规则》。

## 3. 内容编写规则

- 课程标准和教材介绍只用于核对课程边界、概念范围和编排变化。
- 知识讲解、公式说明、示例、步骤和图示均由项目原创整理。
- 用户提供的八年级练习本照片只用于理解“专题 -> 考点 -> 考法”层级，不录入原书题目、解析、正文或插图。
- “手拉手”“一线三等角”“A 型/K 型”等名称是常见教学模型标签，不属于教材正式章节名；页面必须同时给出可验证的几何条件，不能只凭模型名称下结论。
- 模型结论必须写清适用条件，例如直角三角形的 HL、圆外同一点的切线长、三角形中位线和等可能概率。
- 项目不设置学完自测、单元输出任务或测评门槛，数学页面只提供知识、方法和原创示例。

## 4. 专题结构

数学保留“年级大包 -> 专题小包 -> 章节 -> 知识点/方法模板”的结构：

- 七年级 10 个专题：有理数、整式、一次方程、几何初步、平行线、实数、坐标、方程组、不等式、数据收集整理。
- 八年级 10 个专题：三角形、全等、轴对称、整式乘法与因式分解、分式、二次根式、勾股定理、平行四边形、一次函数、数据分析。
- 九年级 9 个专题：一元二次方程、二次函数、旋转、圆、概率、反比例函数、相似、锐角三角函数、投影与视图。

专题使用稳定语义 ID。教材内容拆分、移动或重命名时，优先调整章节映射，不更换用户已经保存的专题 ID。

## 5. 准确性门禁

89 个教材小节的公式与结论集中维护在 data/math-lesson-formulas.js，避免按主题复用公式造成章节错配。知识事实维护在 data/math-lesson-facts.js，36 条旧链接知识仍纳入兼容检查。

修改数学内容后至少执行：

    node scripts/check-math-content.js
    node scripts/check-math-accuracy.js
    node scripts/check-unique-figures.js

逐章复核结果见《数学内容准确性复核记录》。
